# Frontend Harness Slides Live Demo

This repository powers the public Workbench Demo for **Frontend Harness Slides**.

Open the live demo:

https://frontend-harness-slides-demo.vercel.app/

The demo shows a set of dynamic slide style presets with real rendered scenes,
motion beats, density changes, theme switching, and bilingual UI. It is meant as
a visual reference for choosing a direction before building a full slide deck.

For the actual agent skill, installation instructions, references, and examples,
use the standalone skill repository:

https://github.com/patrick-fu/frontend-harness-slides

For Patrick Fu's broader curated skill collection:

https://github.com/patrick-fu/awesome-skills

## Local Development

```bash
npm install
npm run dev
npm test
```

The app is built with React, Vite, Tailwind CSS, and Playwright.
